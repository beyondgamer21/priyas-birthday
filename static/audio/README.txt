# Background Music

Please place your background music file named 'background-music.mp3' in this folder.

The music will play automatically when a user interacts with the website.
Users can toggle the music on/off using the speaker icon in the top right corner.

Recommended features for the music:
- Soft, romantic background music
- No lyrics or minimal lyrics
- Around 2-3 minutes long (it will loop automatically)
- MP3 format for best compatibility
